## Getting Started

### Prerequisites

- Flutter SDK (>=3.2.3)
- Dart SDK (>=3.0.0)
- Android Studio / VS Code
- Android SDK / Xcode (for iOS)

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/todoapp.git
```

2. Navigate to the project directory
```bash
cd todoapp
```

3. Install dependencies
```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```
